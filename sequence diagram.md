### 3. Screen Layout Specification

To ensure high usability on standard 16:9 Android handheld devices, the user interface adheres to strict industrial UX rules.

![Mobile HMI UI Wireframe Layout](./docs/mockups/main_layout.png)

*   **Top 20% Header:** Reserved for real-time telemetry updates.
*   **Action Safety:** `TRANSFER` commands are structurally disabled if the target stocker drops `OFFLINE`.
